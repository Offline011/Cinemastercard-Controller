
Как использовать:

1. Открой сайт
2. Загрузи Excel файл расписания
3. Сайт автоматически создаст список залов и сеансов
4. Нажимай "Проверить зал"
5. Таймер показывает сколько осталось до следующей проверки
6. Галочки:
   - Есть продажи
   - Люди присутствуют

Красный = прошло больше часа
Зеленый = проверка актуальна
